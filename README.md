# tareco
# O que o meu sistema faz?

Este sistema é um mural de recados que permite cadastrar e armazenar mensagens em um banco de dados.
Os usuários podem visualizar os recados de forma simples e organizada

# Como rodar o projeto?

1. Clone o projeto
2. Configure a variável de ambiente na Vercel
3. Importe o banco de dados
4. Execute o projeto

# Se o site cair, o que fazer?

Acesse o painel da Vercel e utilize a opção de Rollback para restaurar a última versão estável do sistema
